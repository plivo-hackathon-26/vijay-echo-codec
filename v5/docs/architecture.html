L1 · Input gate ~0 ms

L2 · Deterministic diff ~µs

Extract claims, diff against ground truth: spoken price vs reference store · "I've cancelled it" vs the tool log · readbacks vs session state. Exact, explainable, no model.

▼no hard mismatch

Assertiveness gate ~µs

Does this reply claim anything? (a number, a commitment, "it's done", "we offer…"). NO → speak immediately, 0 ms added. YES → it pays the judge.

▼assertive (~57% of turns)

⚖️ Grounded LLM judge p50 1.3s · timeout 3.5s

One stateless call: FACTS + POLICIES + recent turns + draft → violation? yes/no + why. Timeout or error → FAIL-OPEN (release; a stuck judge must never freeze a phone call).

🛠 Correction loop

1 · filler speaks NOW: "Let me double-check that…"
2 · violation + verified truth → main LLM regenerates
3 · candidate is re-gated (and must not echo the wrong value)
4 · pass → speak it · 2 fails → safe handoff line

▼after the call — off the hot path

L4 · Post-call judge backstop

Re-audits every agent turn offline. Catches whatever failed open or slipped the gate — nothing is silently lost.
