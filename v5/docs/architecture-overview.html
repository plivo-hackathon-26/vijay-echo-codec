plivo-mirror v5 — grounded output firewall for LLM voice agents

Watches the agent, not the caller. Every claim the agent speaks is diffed against deterministic ground truth held in-stack — session state, structured reference data, the tool log — with a {spoken, truth, source} receipt. One engine, two deployables (shadow monitoring · live intervention), selected by a single mode flag.

L1 — Input Integrity GATE

engine/layers/l1_input_integrity.py · user turns

Not a detector. (1) Low ASR confidence → mark turn untrusted so L2 downgrades to info (don't punish answering a mis-transcribed question). (2) Readback corrections → write the corrected fact into SessionState. All verdicts info.

L2 — Deterministic Diff PRIMARY · µs · inline-safe ≤50ms

engine/layers/l2_deterministic.py + l2_checks.py · agent turns

Claims diff — resolve each claim's referent & compare:

session.* → SessionState reference.* → ReferenceStore tool.* → tool log (speech-vs-action)

Unresolved referent → outside L2 → judge's job.

Parallel policy checks (the v4 defenses, regex/dict, µs):

tool_args → wrong-action authorization → prompt-injection commitments disclosures persona drift

Every verdict carries Evidence {claim_type, spoken_value, truth_value, source} — rendered verbatim in the dashboard.

Arbitration — deterministic wins

engine/arbitration.py

Per claim, precedence L2 > JUDGE. A weaker same-claim verdict stops firing & records suppressed_by=["L2"] — auditable; never two firing verdicts for one claim.

Ground truth (read by L2)

Hook B — pre-TTS gate PREVENTION

deployables/intervention/hook_b_pre_tts.py · integrations/pre_tts.py

Sits between LLM output and TTS — flagged draft never reaches the speaker:

  • 1. L2 deterministic floor (µs) — hard hit holds, no model
  • 2. Assertiveness gate (µs, engine/gate.py) — chitchat/questions release at ~0ms
  • 3. Inline grounded JUDGE — assertive turns only, hard timeout, fail-open

Hold → CorrectionRetryLoop: filler → regenerate(main LLM w/ packet) → re-gate + pink-elephant echo check → cap retries → safe handoff.
